How to Use the Potentiometers

The hardware controller has three circle potentiometers, or “pots”. all the pots work in the same fashion. They are used to control an audio effect and a Paintbrush effect at the same time. Turn the knobs to control the effect levels.

Pot 1 : this pot is used to control the reverb level, and also affects the width of the paintbrush stroke.

Pot 2 : this pot is used to control the distortion level of the audio, and simultaneously affects the jitter of the paintbrush.

Pot 3 : the third pot is used to control a high-pass filter level, and is also responsible for the color saturation level.
